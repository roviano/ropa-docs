## What the Integrator Package Adds

The Integrator Package includes **everything in the Developer Kit** plus four production-critical additions:

| Developer Kit includes | Integrator adds |
|------------------------|----------------|
| Chassis (any iMRP model) | **On-site deployment assistance** |
| ROPA SDK + full documentation | **Priority engineer support** (48h response SLA) |
| Python / JS / Android examples | **Integration recipe cookbook** |
| 1-on-1 engineer support (remote, 72h response) | **Production readiness audit** |

### What Each Addition Means

#### On-Site Deployment Assistance

Our engineer arrives at your deployment site to:
- Verify physical environment (floor conditions, obstacle layout, elevator compatibility)
- Complete the first map build and POI configuration
- Test navigation routes under real conditions
- Verify charging dock placement and auto-dock reliability
- For iMRP600: configure elevator controller integration for multi-floor deployments

#### Priority Engineer Support (48h SLA)

- Response within **48 hours** (vs 72h for Developer Kit)
- Direct access to senior integration engineer
- Covers: protocol troubleshooting, map optimization, multi-robot coordination, custom hardware integration guidance

#### Integration Recipe Cookbook

Pre-written deployment recipes for common scenarios:
- Hospital delivery (multi-floor, iMRP600)
- Warehouse inventory shuttle (iMRP500/800)
- Hotel concierge patrol (iMRP400/500)
- Factory material transport (iMRP800)

Each recipe includes: POI layout, speed/travel settings, map editing zones, monitoring dashboard config, and production checklist.

#### Production Readiness Audit

Before you go live, our engineer reviews:
- Map quality and coverage completeness
- POI accuracy and naming conventions
- Safety zone configuration (virtual walls, deceleration zones)
- Charging dock reliability
- Localization confidence across all routes
- Emergency stop accessibility
- Battery endurance under production load profile

You receive a **signed production readiness report** confirming the system meets deployment requirements.

## Integrator Package Workflow

### Phase 1: Pre-Deployment (Remote)

1. You receive the Developer Kit components (chassis + SDK + docs)
2. Build your application software using ROPA protocol
3. Test in your development environment
4. Schedule on-site deployment date with our engineer

### Phase 2: On-Site Deployment (1–2 Days)

| Day | Activities |
|-----|-----------|
| **Day 1** | Environment survey → first map build → POI configuration → initial route testing |
| **Day 2** (if needed) | Elevator configuration (iMRP600) → safety zone edits → charging dock verification → multi-route testing |

### Phase 3: Production Readiness Audit

- Engineer reviews all configurations against production checklist
- Signs production readiness report
- Hands over complete system documentation

### Phase 4: Production Operation

- You operate the deployed system
- Priority support available (48h SLA) for any issues
